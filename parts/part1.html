Setting up

First: Getting LOVE installed. There is a guide for that at its wiki. Once you've got it working, create a directory for your game. Follow the wiki's "Making a Game" section and make sure it prints to the screen properly. Next, we will be using the fantastic rotLove library for various parts of the game, including our display. I have made some slight modifications to it that allow us to use our own tileset. You can download it here or use git:

git clone https://github.com/LJNIC/rotLove.git

Make a subdirectory in your game directory called something like "lib". Take the "src" from rotLove and stick it in lib. If you haven't already, open up a file in your text editor of choice and call it "main.lua". This is where all LOVE games begin. Our code will start off like this:
